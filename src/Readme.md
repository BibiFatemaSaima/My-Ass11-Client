# 🚍 TicketBari - Online Ticket Booking Platform

TicketBari is a modern MERN Stack based online ticket booking platform where users can easily browse, book, and pay for Bus, Train, Launch, and Flight tickets. The platform supports three different roles: User, Vendor, and Admin, ensuring smooth ticket management and booking operations.

## 📦 NPM Packages

### Client Side

```bash
npm install react-router-dom

npm install axios
npm install react-icons
npm install react-hot-toast
npm install sweetalert2
npm install swiper
npm install recharts
npm install @stripe/react-stripe-js
npm install @stripe/stripe-js
```

### Server Side

```bash
npm install express
npm install mongodb
npm install cors
npm install dotenv
npm install jsonwebtoken
npm install stripe
```
